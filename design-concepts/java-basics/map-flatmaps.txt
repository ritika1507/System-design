Difference between map() and flatMap()?

map() â†’ transforms each element into one element.
flatMap() â†’ transforms each element into multiple elements and flattens.
List<String> words = Arrays.asList("Hello", "World");

// map()
List<String[]> mapped = words.stream()
                             .map(word -> word.split(""))
                             .toList(); 
// [[H, e, l, l, o], [W, o, r, l, d]]

// flatMap()
List<String> flatMapped = words.stream()
                               .flatMap(word -> Arrays.stream(word.split("")))
                               .toList();
// [H, e, l, l, o, W, o, r, l, d]
